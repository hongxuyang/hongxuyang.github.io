函数
是指可重复使用的程序片段，允许你为某个代码块赋予名称，允许你通过这个特殊的名字在你的程序任何地方来运行代码块，并可重复任何次数。这就是所谓的调用函数。之前我们已经学过了内置函数len和range

通常函数可以用def来定义，这一关键字后跟一个函数的标识符名称，再跟一圆括号，其中可以包括一些变量的名称，再以冒号结尾
举个例子：
def say_hello():——定义一个函数

print(‘hello world’)——函数会返回的内容

say_hello()——下面都是调用函数
say_hello()
say_hello()

函数参数
在定义函数时给定的名称称作为“形参”，在调用的时候你所提供给函数值称为“实参”
def print_max(a,b):——定义函数
if a>b:——写条件语句
print(a,’is maximum’)
elif a==b:
print(a,’ is equal to ‘,b)
else:
print(b,’is maximum’)
print_max(3,4)——直接传递字面值，以实参的形式直接向函数提供这一数字
如果不想用这种调用方式，可以使用如下的：
x=5
y=6
print_max(x,y) ——将上面的最后一行，换成这三行就行了，使得实参x的值和y的值分别被赋值给形参a，b。

局部变量
当你在一个函数的定义中声明变量时，它们不会以任何方式与身处函数之外但具有相同名称的变量产生关系，也就是说吗，这些变量名只存在于函数这一局部。这被称为变量的作用域。所以变量的作用域是它们被定义的块，从定义它们名字的定义点开始。
x=50
def func(x):
print(‘x is , ‘,x)
x=2
print(‘changed local x to ‘,x)
func(x)
print(‘x is still ‘,x)
会输出：
x is , 50
changed local x to 2
x is still 50
如何工作的呢？
第一次打印的是主代码块上x的值，接着将2赋值给x，x是我们这函数的局部变量。因此当我们改变这个x的时候，上个x即主代码块中没变化。最后的print是主代码块中定义的x的值，所以他是不会受局部变量影响的

Global语句
可以为一个在程序顶层的变量赋值，其实也就是说他不存在任何作用域中，无论函数还是类，那么你必须告诉python这个变量是非局部的，而且全局的，global语句就可以做到。注意：不适用global时，不可能为一个定义于函数之外的变量赋值。通过使用 global 语句便可清楚看出这一变量是在最外边的代码块中定义的
x=50
def func():
global x
print(‘x is , ‘, x)
x = 2
print(‘changed local x to ‘, x)
func()
print(‘value of x is ‘, x)
会输出：
x is , 50
changed local x to 2
value of x is 2
代码与上面做的局部变量的实验相似。但是不同的是global语句用以表明x是一个全局变量，因此当我们在函数中进行为x赋值时，这一影响将改动主代码块中的x的值。
你可以在global语句中定义不止一个全局变量。例如global x,y,z

其实 全局变量就是，只要你改，他后面都跟着改，不像局部变量，只会在你改动的那部分改动它的值！

默认参数值
其实默认参数值应该就是常熟，更确切的说，默认参数值应该是不可变的。
def say(message,times=1):
print(message*times)
say(‘Hello’)
say(‘World’,5)
会输出：
Hello
WorldWorldWorldWorldWorld
如何工作？
Say函数定义好次数从而来打印一串字符串，如果没这个数值就默认只打印一次。通过参数times指定默认参数值1来实现这一点
所以可以推断是怎么打印的了吧！

注意：只有那些位于参数列表末尾的参数才能被赋予默认参数值，意即在函数的参数列表中拥有默认参数值的参数不能位于没有默认参数值的参数之前。
因为值是按参数所处的位置依次分配的。举例来说，def func(a, b=5) 是有效的，
但 def func(a=5, b) 是无效的。

关键字参数
如果你有一些具有许多参数的函数，而你又希望只对其中的一些进行指定，那么你可以通过命名它们来给这些参数赋值——这就是关键字参数，我们使用命名（关键字）而非位置（一直以来我们所使用的方式）来指定函数中的参数
两大优点：其一、我们不用考虑参数的顺序，函数的使用将更加容易。其二、可以只对那些我们希望赋予的参数以赋值，只要其它的参数都具有默认参数值
def func(a,b=5,c=10):
print(‘a is’,a,’and b is’,b,’and c is’,c)
func(3,7)
func(25,c=24)
func(c=50,a=100)
会输出：
a is 3 and b is 7 and c is 10
a is 25 and b is 5 and c is 24
a is 100 and b is 5 and c is 50
简单解释下，func函数中有一个没有默认参数值的参数，后跟两个各自带有默认参数值的参数。
第一次调用：a获得3，b获得7，c使用了默认
第二次调用：a获得25，c被指定获得24
第三次调用：全部使用关键字参数来指定值。尽管a在c之前定义，但我们还在变量a之前定义了变量c

可变参数
有时你想定义的函数里能有任意数量的变量，也就是参数数量是可变的，这可通过星号来实现
def total(a=5,numbers,*phonebook):
print(‘a’,a)
for single_item in numbers:——遍历元组中的所有项目
print(‘single_item’,single_item)

for first_part,second_part in phonebook.items():——遍历字典中的所以项目
    print(first_part,second_part)

print(total(10,1,2,3,Jack=1123,John=2231,Inge=1560))
会输出：
a 10
single_item 1
single_item 2
single_item 3
Jack 1123
John 2231
Inge 1560
None
如何工作的呢？
声明一个诸如param的星号参数时，从开始到结束所以位置都将被收集并汇聚层一个称为param的元组
类似地，当声明一个诸如*param双星号参数时，从此处开始直至所以结束的所以关键字参数都将被收集并汇聚为一个名为param的字典

Return
用于从函数中返回，也中断函数
def maximum(x,y):
if x>y:
return x
elif x==y:
return ‘数值相等’
else:
return y
print(maximum(2,3))
当然会输出3
定义的Maximum函数会返回参数中的最大值，本例中是提供给函数的数值，使用if…else来判断最大值。
要注意如果return没有搭配任何一个值代表返回none。在python中代表虚无。
每个函数都在末尾隐含了一句return none 除非你自己写的return语句。你可以运行print(some_function())，其中some_function函数不适用return语句。如下：
def some_function():
pass

docstrings
文档字符串。能够帮你更好地记录程序并让其更加容易于理解。而且可以当程序运行时，可以通过一个函数来获取文档。
def print_max(x,y): ——打印这两个数值中的最大值

x=int(x) ——如果这两个都应该是整数
y=int(y)

if x>y:
    print(x,'is maximum')
else:
    print(y,'is maximum')

print_max(3,5)
print(print_max.doc)
会输出：
5 is maximum
None
为什么呢？
函数的第一行逻辑中的字符串是该函数的文档字符串。这里注意文档字符串也适用于模块和类。
文档字符串的惯例是一个多行字符串，它的首行以大写字母开始，句号结尾。第二行是空行，从第三行开始是详细的描述。 强烈建议 你在你的函数中使用文档字符串时遵循这个惯例。
你可以使用doc（注意双下划线）调用printMax函数的文档字符串属性（属于函数的名称）。请记住Python把 每一样东西 都作为对象，包括这个函数。我们会在后面的类一章学习更多关于对象的知识。
如果你已经在Python中使用过help()，那么你已经看到过DocStings的使用了！它所做的只是抓取函数的doc属性，然后整洁地展示给你。你可以对上面这个函数尝试一下——只是在你的程序中包括help(printMax)。记住按q退出help。
自动化工具也可以以同样的方式从你的程序中提取文档。因此，我 强烈建议 你对你所写的任何正式函数编写文档字符串。随你的Python发行版附带的pydoc命令，与help()类似地使用DocStrings。
