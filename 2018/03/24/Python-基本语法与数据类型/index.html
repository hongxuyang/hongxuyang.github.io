a=’abcde’

想要取多个？

简单介绍下切片：[索引值：结束值+1]

a[1:4]
‘bcd’
如果没有索引值，或者结束值，代表从头取或者取到结尾
补偿值的使用：代表走几步取，如下两例
a[::1] 【从头到尾，走一步就取】
‘abcde’
a[::2]
‘ace’
当然还可以用负数来取：
a[-1] 【代表取最后一个】
‘e’
也可以像正数那样取，[起始点：结束点]，注意，结束点是不包含的！！！而且一定是从左往右的。但是一般不这样取，一般用正数。
a[-4:-1]
‘bcd’
如果想用负数一个个取，可以使用补偿值！【当补偿值为1时代表从左往右，为-1时代表从右往左】
a[-2:-4:-1]
‘dc’

字典
Python中唯一的一个映射类型，存储的是哈希表。对象是可变的，但是字典的键必须使用不可变对象，并且一个字典中可以使用不同类型的键值

如何创建：
Dic={key:value,key:value}
例：dic={‘name’:’milo’,’age’:20,’gender’:’male’}
如何取值？

显然这样是不行的，在字典里是必须输入他的key值来取值的。如下：
dic[‘name’]

内建方法：fromkeys()，字典中的元素具有相同的值，默认为None
